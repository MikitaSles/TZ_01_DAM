# Подход 1 — Расчёт TVL через Etherscan

## 1. Получить asset()
На вкладке “Read as Proxy” вызываем метод `asset()`.  
Это адрес базового токена (ERC‑20).

## 2. Получить totalAssets()
В том же разделе вызываем `totalAssets()`.  
Получаем сырое значение TVL в минимальных единицах USDC.

## 3. Узнать decimals
Переходим на страницу ERC‑20 токена.  
Метод `decimals()` возвращает 6 — стандарт для USDC.

## 4. Пересчитать в человекочитаемый формат
```
TVL = totalAssets / 10^6
```

## 5. Пример
```
totalAssets = 6175400601224
TVL = 6,175,400.601224 USDC
```

# Подход 2 — Расчёт TVL через Web3.py

*Файл:* `rpc_method.py`

Используется минимальный ABI (только функции `asset`, `totalAssets`, `decimals`, `symbol`).

Порядок:

1. Получить адрес базового актива (`asset()`).
2. Считать метаданные токена (`decimals`, `symbol`).
3. Вызвать `totalAssets()` с RPC.
4. Посчитать TVL.

Код полностью воспроизводит ручной процесс, но автоматизирован.
