# Критичная метрика помимо TVL : цена доли волта 

### Share price (PPS)

Она показывает реальную доходность волта, в отличие от TVL, который отражает масштаб и может меняться из‑за притоков/оттоков.

## Формула (ERC-4626)

totalAssets() - общий объём базовых активов волта,
totalSupply() - количество выпущенных долей волта.

```
Share price (PPS) = totalAssets()/totalSupply()
```

# Практическая часть 

Практическая часть сбора метрик состоит из несольких файлов :
`db.py`,
`GetMetricScript.py`,
`models.py` .

Значение :
`db.py` - подключение к БД.
`GetMetricScript.py` - скрипт для выгрузки, трансформации и загрузки данных в БД.
`models.py` - схема данных для создания таблиц, описываем структуру таблиц.
